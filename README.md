## CircleProgressBar
<br />
Легкий и простой анимированный прогресс бар.<br /><br />
Имеет всего два метода: установоить значение и сбросить до 0.<br />
При инициализации дополнительно можно выбрать цвет и ширину линии.<br /><br />
